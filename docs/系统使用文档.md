# 数据分类分级系统 - 用户手册

## 一、系统简介

本系统是一个基于大语言模型（DeepSeek-7B）的数据字段分类分级工具，支持：

- ✅ 自动识别字段类型（24类）
- ✅ 自动判断数据敏感级别（4级）
- ✅ 支持多行业数据（金融、医疗、教育、商业、工业等）
- ✅ 提供交互式和批量预测两种模式

## 七、API 调用

### 7.1 Python API

```python
from scripts.predict import load_model, build_prompt, predict

# 加载模型
model, tokenizer = load_model()

# 构建提示
prompt = build_prompt(
    industry="金融",
    column_name="客户年龄",
    sample_values=["25", "35", "45", "55"],
    task="分级"
)

# 预测
result = predict(model, tokenizer, prompt)
print(f"预测结果: {result}")
```

### 7.2 批量处理

```python
import pandas as pd
from scripts.predict import load_model, build_prompt, predict

# 加载模型
model, tokenizer = load_model()

# 读取数据
df = pd.read_csv("data/your_file.csv")

# 批量预测
results = []
for col in df.columns:
    prompt = build_prompt("通用", col, df[col].head(5).tolist())
    result = predict(model, tokenizer, prompt)
    results.append({"字段": col, "分类": result})

print(results)
```

## 九、常见问题

### Q1: 显存不足怎么办？

**解决方案**：
1. 减小批次大小：`per_device_train_batch_size=1`
2. 使用梯度累积：`gradient_accumulation_steps=8`
3. 使用 CPU 推理（仅用于测试）

### Q2: 预测结果不准确？

**可能原因**：
1. 样本值不够典型
2. 字段名不清晰
3. 行业选择错误

**解决方案**：
1. 提供更多样本值
2. 使用英文行业缩写
3. 检查字段语义

### Q3: 如何添加新的数据集？

**步骤**：
1. 将 CSV 文件放入 `data/raw/` 目录
2. 在 `batch_convert_datasets.py` 中添加字段标签映射
3. 运行 `python scripts/batch_convert_datasets.py`

### Q4: 如何添加新的分类标签？

**步骤**：
1. 在 `data/labels/classification_schema.json` 中添加新标签
2. 在 `scripts/batch_convert_datasets.py` 的 `CLASSIFICATION_LABELS` 中添加
3. 重新生成训练数据
